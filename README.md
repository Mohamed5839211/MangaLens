# 📸 MangaLens (مانجا لنس)

تطبيق ذكي ومتقدم للهواتف المحمولة يتيح قراءة وترجمة المانغا والمانهوا والمانها (اليابانية، الكورية، الصينية، والإنجليزية) في الوقت الفعلي إلى اللغة العربية الفصحى على الجهاز مباشرة. يدمج التطبيق بين متصفح ويب متطور مانع للإعلانات ونظام متكامل للرؤية الحاسوبية (Computer Vision) والترجمة الفورية باستخدام الذكاء الاصطناعي مع الحفاظ على جماليات الرسوم الأصلية (Torii Image Translator Style).

---

## 🌟 الميزات الرئيسية (Key Features)

### 1. 🛡️ متصفح مانع للإعلانات والتعقب (Ad-Blocked In-App Browser)
- **حظر شامل للإعلانات:** تصفح آمن ونظيف لمواقع المانغا باستخدام فلترة شبكية صارمة (مثل EasyList) لحظر النوافذ المنبثقة (Pop-ups) وإعلانات البوب-أندر (Pop-under) والتعقب.
- **توجيه الروابط الذكي (Same-host Redirect Bypass):** اعتراض روابط الفتح التلقائي للفصول وتصفحها داخل نفس التبويب لمنع إعادة التوجيه لصفحات إعلانية خارجية.
- **إدارة التبويبات المتعددة:** حفظ التبويبات تلقائياً (عددها، روابطها، التبويب النشط) واستعادتها بشكل آمن عند إعادة تشغيل التطبيق، مع عرض بطاقات معاينة مرئية لكل تبويب (Screenshot Previews).

### 2. 🔍 محرك التعرف الضوئي على النصوص (On-Device OCR)
- **معالجة متوازية على الجهاز:** استخدام **Google ML Kit** للتعرف على النصوص بأربع لغات مختلفة بالتوازي (اليابانية، الكورية، الصينية، والإنجليزية).
- **كشف دقيق للهياكل:** استخراج النصوص ومواقعها الجغرافية بدقة عالية (Bounding Boxes) مقسمة على مستوى الفقاعة أو الأسطر الفردية.

### 3. 🧹 تنظيف الفقاعات الذكي (Bilinear Pixel-level Inpainting)
- **مسح النصوص الأصلية:** استخدام مكتبة **OpenCV** عبر خوارزمية `INPAINT_TELEA` لمسح النصوص الأصلية تماماً من الفقاعات وملء الخلفية بلون وتدرج متناسق.
- **التمدد الديناميكي (Dynamic Dilation):** احتساب تمدد القناع تلقائياً بناءً على حجم الخط لحماية الحدود الحساسة للفقاعات وإزالة الظلال الخارجية فقط.
- **نظام طوارئ ذكي (Bilinear Fallback):** استبدال تلقائي لخوارزمية OpenCV في البيئات غير المدعومة عبر معالجة بكسلية ثنائية الأبعاد (Bilinear Local Interpolation) تعمل بالكامل في Isolate مستقل لضمان بقاء التطبيق بـ 60 إطاراً في الثانية.

### 4. 🤖 الترجمة الفورية والتبديل التلقائي (Groq API & Model Rotation)
- **تعريب احترافي متناسق:** ربط مباشر بـ Groq API (عبر موديل Llama 3.3 70B Versatile) لتعريب النصوص مع الحفاظ على سياق القصة، نبرة الشخصيات، واختصارات الـ RPG.
- **تجاوز حظر المحتوى والحدود (Model Rotation):** تدوير ذكي تلقائي بين عدة موديلات بديلة عند مواجهة ضغط طلبات (Rate Limit 429) أو عند كشف تسريب أحرف غير مترجمة، لضمان استمرارية الخدمة.
- **تسمية مخصصة وعلامات صوتية:** تعريب أسماء الشخصيات ووضعها بين قوسين تلقائياً وتحديد المؤثرات الصوتية بوسوم مخصصة `[SFX: ...]` لتنسيقها بشكل مختلف.

### 5. ✍️ عارض النصوص الاحترافي (RTL Text Renderer)
- **تنسيق ذكي ثلاثي الأنماط:**
  - **فقاعات المانغا اليابانية:** توسيع أفقي ذكي للفقاعات الرأسية لتستوعب النصوص العربية الأفقية دون تصغير حجم الخط.
  - **نوافذ إحصائيات النظام (RPG Status Windows):** كشف وتصميم رقمي تقني نيون للوحات الإحصائيات مع الحفاظ على توزيع الأسطر الداخلي.
  - **بوابات السفر عبر الزمن وإشعارات النظام (Gate Alerts):** إطار ذهبي وتصميم غامض يناسب المانهوا القتالية والحديثة.
- **خطوط عربية متميزة:** استخدام Cairo و Amiri للرسم المتناسق مع دعم الخطوط اللاتينية الفاخرة للغات الهدف الأخرى.

## 🛠️ البنية التقنية (Tech Stack)

- **الواجهة والتطبيق الأساسي:** Flutter & Dart.
- **إدارة الحالة (State Management):** Flutter Riverpod.
- **الرؤية الحاسوبية:** OpenCV Dart & Google ML Kit Text Recognition.
- **الترجمة:** Groq Cloud API & OpenAI API Client (Dio).
- **التخزين الآمن:** Flutter Secure Storage.
- **محرك الرسم:** Flutter Canvas (`ui.PictureRecorder`).

## 🚀 تشغيل وتجهيز التطبيق (Installation & Run)

### المتطلبات الأساسية:
- Flutter SDK (v3.22.0 أو أحدث).
- Android SDK / Xcode لتشغيل محاكي الهواتف الذكية.
- مفتاح API الخاص بـ **Groq** أو **OpenAI** (يتم إدخاله في إعدادات التطبيق أو عبر ملف `.env`).

### خطوات التشغيل:

1. **نسخ المشروع:**
   ```bash
   git clone https://github.com/your-username/MangaLens.git
   cd MangaLens
   ```

2. **تثبيت الحزم البرمجية:**
   ```bash
   flutter pub get
   ```

3. **تهيئة ملف البيئة (اختياري):**
   قم بإنشاء ملف باسم `.env` في المجلد الرئيسي للمشروع، وأضف مفتاحك الافتراضي:
   ```env
   DEFAULT_GROQ_API_KEY=gsk_your_groq_api_key_here
   ```

4. **تشغيل الفحص البرمجي للتأكد من سلامة الأكواد:**
   ```bash
   flutter analyze
   ```

5. **تشغيل التطبيق على المحاكي أو الهاتف:**
   ```bash
   flutter run
   ```
